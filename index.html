How to Use

使用方法

1

Install the Extension

拡張機能をインストール

Get VSCode Game Sound from the VS Code Marketplace

VS Codeマーケットプレイスから「VSCode Game Sound」をインストール

2

Start Coding

コーディングを開始

Open any file and start typing to hear the game sounds

任意のファイルを開いてタイプすると、ゲームサウンドが鳴ります

3

Customize Settings

設定をカスタマイズ

Open Settings and search for "Game Sound" to customize the extension

設定を開き、「Game Sound」を検索して拡張機能をカスタマイズ

4

Enjoy Game-like Coding

ゲーム風コーディングを楽しもう

Experience the fun of programming with game-like sound effects!

ゲーム風効果音でプログラミングを楽しく体験しましょう！

Configuration

設定オプション

Setting設定
Description説明
gameSound.enabled
Enable or disable all game sounds ゲームサウンドの有効/無効を切り替え
gameSound.volume
Sound volume (0-100) 音量（0-100）
gameSound.keyCooldown
Minimum time between sounds for the same key (milliseconds) 同一キーの再生間隔（ミリ秒）
gameSound.specialKeys
Enable or disable sounds for specific keys 特定のキーのサウンド有効/無効設定
